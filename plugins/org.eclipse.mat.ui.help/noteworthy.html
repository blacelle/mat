New & Noteworthy

Here are the descriptions of some of the more interesting or significant changes made to Eclipse Memory Analyzer.

Unreachable objects

By default unreachable objects are removed from the heap dump while parsing and will not appear in class histogram, dominator tree, etc. Yet it is possible to open a histogram of unreachable objects via "Java Basics --> Unreachable Objects Histogram" query. For more details please check our wiki page http://wiki.eclipse.org/MemoryAnalyzer/FAQ#How_to_analyse_unreachable_objects

Inspector View Enhancements

When analyzing a dump from a SWT application, the inspector recreates colors and images and displays the information in the lower left corner.
