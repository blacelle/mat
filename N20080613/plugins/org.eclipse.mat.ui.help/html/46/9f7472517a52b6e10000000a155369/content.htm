Object Query Language (OQL)

The Memory Analyzer has a built-in object query language (OQL) that allows it to query the heap dump with custom SQL-like queries. Similarly to SQL, you should think of classes as tables, objects as rows, and fields as columns. The basic syntax is as follows:

SELECT *

FROM [ INSTANCEOF ] <class name>

[ WHERE <filter-expression> ]
