Class References View

The Class References view groups the relationship information of objects by classes.

You can reach the classes using the context menu Show class references...From/To.

“From this class” means that all objects referenced by objects of the source class are analyzed and grouped by classes.

“To this class” means that all objects referencing objects of the source class are analyzed and grouped by classes.

In the screenshot above, you see that String objects are mostly referenced by Hashtable$Entry objects which are in turn mostly referenced by the corresponding Hashtable$Entry arrays and objects. You can open and browse the tree.
